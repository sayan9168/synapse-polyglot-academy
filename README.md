**Synapse Academy** is an all-in-one, AI-powered learning ecosystem designed to simplify complex engineering concepts. From high-level web development to low-level kernel architecture, we bridge the gap with AI-human collaboration.

## 🚀 Key Features

- **Multi-Language Web Notebook:** Write and run code for 50+ languages (Python, Rust, C++, Aion, etc.) directly in your browser.
- **AI Easy-Mode Tutor:** Get real-time, simplified explanations for complex code snippets powered by Gemini AI.
- **Extreme Tier Curriculum:** Exclusive tracks for Cybersecurity, Custom Language Design, and SUG Theory.
- **Automated Certification:** Earn a professional, QR-verified certificate signed by **Sayan Mahata** upon course completion.
- **Global Payment Gateway:** Seamless access to premium content from anywhere in the world.

## 🏗️ The Curriculum (Extreme Level)

| Track | Focus Area | Key Projects |
| :--- | :--- | :--- |
| **Low-Level Dev** | Kernel & Compiler Design | Aion Language Core |
| **Cybersecurity** | Offensive Security & Pentesting | OmniSentinel & ApexSentinel |
| **AI Engineering** | LLM Integration & Automation | NovaQL & MindSQL |
| **Scientific Theory** | Hardware-Logic Intersection | SUG Theory (RHP) |

## 🛠️ Tech Stack

- **Frontend:** Next.js, Tailwind CSS, Monaco Editor
- **Backend:** Node.js, Piston API (Execution), Gemini API (AI Tutor)
- **Deployment:** Vercel / Docker
- **Security:** JWT Authentication & Rate Limiting
