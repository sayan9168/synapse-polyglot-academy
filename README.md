# 🌌 Synapse Polyglot Academy

Synapse is a next-generation education platform designed to teach **any** computer language—from Python to Rust, and even custom hardware-level languages. 

## ✨ Features
- **All Languages Free:** Learn the syntax and basics of 50+ languages at no cost.
- **Interactive Notebook:** In-browser code execution engine to test scripts instantly.
- **Extreme Level (Paid):** Unlock deep-dive system architecture, low-level optimization, and AI-assisted debugging.
- **Global Payments:** Supported across 135+ currencies for global accessibility.

## 🛠️ Tech Stack
- **Frontend:** Next.js, Tailwind CSS, Monaco Editor
- **Backend:** Node.js, Dockerized Execution Engine
- **Database:** PostgreSQL (with NovaQL logic)
- **Payments:** Stripe & PayPal Global

## 🛡️ Security
Fully isolated code execution using secure containers to prevent system-level vulnerabilities.
